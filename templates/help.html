Posting Help

Posting:

Tripcodes

Entering ## followed by a password will result in a hashed tripcode being added after your name.

Example: Anonymous##example → Anonymous##00fb39c28d

Role Signature

Entering ## followed by rs into the name field instead of a tripcode will result in your role being used

Example: Anonymous##rs → Anonymous##administrator

Text Markdown:

Italics

[i]italicized text[/i] → italicized text

*italicized* → italicized text

Bold Text

[b]bold text[/b] → bold text

**bold text** → bold text

Underlined Text

[u]underlined text[/u] → underlined text

__underlined text__ → underlined text

Strikethrough

[s]strikethrough[/s] → strikethrough

~~strikethrough~~ → strikethrough

Spoiler

[spoiler]spoiler[/spoiler] → spoiler

||spoiler|| → spoiler

Greentext

>greentext → >greentext

Pinktext

<pinktext → <pinktext

Code

[code]code[/code] → code

Rainbow Text

##rainbow text## → rainbow text

Red Text

==red text== → red text

Detected Text

(((detected))) → detected
